The Project

Xenblocks Foundation exists to help people learn about crypto-currencies. We want make it as easy as possible for everyone to learn hands on how crypto currencies work without cost and hassle free.

Features

  • Litecoin clone
  • Windows, Mac, Linux and Web wallets
  • Free coins at sign up and daily give aways
  • Tutorials and documentation
  • An active community

Community

The community is at the center of the XenBlocks project. We want to foster a friendly community of crypto-currencies advocates that want to help drive crypto currencies adoption in the world.
