How to mine Xenblocks


Mining Xenblocks Coins is very simple. You will need to follow these steps:

  1. Get a Xenblocks Address (Note: Xenblocks addresses start with X)
  2. Get the mining software for your OS & hardware
  3. Run the mining program with the settings to join our mining pool and using your Xenblocks Address where the coins will get deposited


Instructions

1. You can get a Xenblocks address by registering at https://wallet.xenblocks.com or by downloading one of the Xenblocks-QT wallets.

a) If you are using the Web Wallet, your Xenblocks Address is the XEN Address value found in the Receive Tab:



b) If you are using one of the Xenblocks-QT wallets, your Xenblocks address is found on the Receive tab on the Address column. You can create new addresses and use any of them.



2. You can get mining software for your OS from here:

  • CPU Mining (most common): pooler-cpuminer
  • NVIDIA GPU Mining (advanced): ccminer
  • AMD GPU Mining (advanced): cgminer


3. To start mining you must excute the software you downloaded with one of the following settings, depending how you decided to mine:

For CPU Mining (pooler-cpuminer):
minerd -a scrypt -o stratum+tcp://pool.xenblocks-coin.org:3008 -u YOUR_XENTAVO_ADDRESS -p random

For NVIDIA GPU Mining:
ccminer-x64 -a scrypt -o stratum+tcp://pool.xenblocks-coin.org:3008 -u YOUR_XENTAVO_ADDRESS -p random

For AMD GPU Mining:
cgminer.exe --scrypt --url=stratum+tcp://pool.xenblocks-coin.org:3008 --YOUR_XENTAVO_ADDRESS:random

Note: You must replace YOUR_XENTAVO_ADDRESS with your own XEN Address found on the Receive tab of the online wallet, or any XEN Address that you have created to receive XEN coins. The pool pays about once a day.



Check your mining status

Once you run your miner, you will see an screen that looks like this:

Every time that you see a yay!! means that your block got accepted and you will earn some XEN coins. The pool pays once a day, so you can except to receive XEN coins within 24 hours after you start mining.



Pool statistics

You can view the status of your mining worker by looking at the pool stats. Find your XEN Address to see the Hashrate of your worker and how many shares you are getting.
